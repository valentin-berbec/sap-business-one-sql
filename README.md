# SAP Business One SQL Query Library

This repository contains SQL queries and reporting examples for SAP Business One (SQL Server version).

The purpose is to demonstrate how ERP data can be analysed to support business decisions in areas such as sales, inventory, and finance.

## Topics covered

- Sales reporting
- Purchasing analysis
- Inventory analysis
- Financial reporting
- SAP Business One alerts and monitoring queries
- SAP Business One table structure

## Repository Structure

- **Alerts/**  
  Example SQL alerts used in SAP Business One Query Manager.  
  These queries highlight potential issues such as low margins or unusual transactions.

- **Finance/**  
  Queries related to financial reporting and analysis, such as invoice summaries or customer balances.

- **Inventory/**  
  Queries analysing stock movements, returns, and inventory activity.

- **Purchasing/**  
  Purchase order reporting queries, supplier analysis, and procurement data extraction.

- **Sales/**  
  Queries demonstrating sales document flows and revenue analysis.

- **Master_Data/**  
  Examples of working with business partner and item master data.

- **Documentation/**  
  Reference material explaining SAP Business One tables, document flow, and reporting concepts.

## Environment

SAP Business One (SQL Server version)  
SQL Server Management Studio (SSMS)
