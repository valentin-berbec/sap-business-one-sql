# SAP Business One Core Tables

## Marketing Documents
- ORDR - Sales Order header
- RDR1 - Sales Order rows
- ODLN - Delivery header
- DLN1 - Delivery rows
- OINV - A/R Invoice header
- INV1 - A/R Invoice rows
- ORIN - A/R Credit Note header
- RIN1 - A/R Credit Note rows

## Business Partners
- OCRD - Business Partner master data

## Items and Inventory
- OITM - Item master
- OITW - Item by warehouse
- OITB - Item groups

## Purchasing
- OPOR - Purchase Order header
- POR1 - Purchase Order rows
- OPDN - GRPO header
- PDN1 - GRPO rows
- OPCH - A/P Invoice header
- PCH1 - A/P Invoice rows

## Useful Link Fields
- DocEntry - internal document key
- DocNum - visible document number
- LineNum - row number
- BaseEntry - source document DocEntry
- BaseLine - source document row
- BaseType - source document object type
